# popl-project
Project Idea: Writing an interpreter that can execute basic mathematical expressions written in a python-style syntax. Possibly extend this to include some special operations such as finding the square root of a number with the notation: ''x

In the interpreter, that becomes:
1.	Define your language data structures: tokens, AST nodes, values
2.	Write example programs and expected results
3.	Design methods by following the structure of the data
4.	Implement parser and evaluator one piece at a time
5.	Test every small part before adding features
